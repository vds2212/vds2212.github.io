# Massive Edition

## Normal

`:normal` macrostring

Remark: the macrostring must be typed literally. The special character like <CR> or <Esc> must be inserted using the
<kbd>Ctrl v</kbd> trick (i.e. <kbd>Ctrl v</kbd><kbd>Esc</kbd> for <Esc>) 

`:normal @a`
